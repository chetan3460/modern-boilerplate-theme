## Implementation Steps

1. Go to WordPress Admin Dashboard
2. Navigate to Pages â†’ Home (or create new page with template `page-home.php`)
3. Scroll to **Home Panels** flexible content field
4. Add layouts in order using examples above
5. Fill in text content as specified
6. Upload images (use WebP format where possible)
7. Set CTAs to appropriate links
8. Set all **Hide Block** toggles to OFF
9. Publish/Update

---

## Tips

- Keep titles under 60 characters for best display
- Use compelling descriptions (2-3 sentences)
- Ensure images are optimized before upload
- Test responsive layout on mobile devices
- Use consistent branding in images
- Verify all CTA links work before publishing
